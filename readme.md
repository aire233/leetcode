这个仓库用于记录我在 LeetCode 上刷题的过程和解题思路。

## 目录结构

在每个题目文件夹中，包含以下全部或部分内容：

- **`README.md`**: 包含了题目描述、示例、解题思路和复杂度分析。
- **解题代码**: 使用多种编程语言实现的解题代码，以及可能的优化版本。
- **测试用例**: 用于验证解题代码正确性的测试用例。

## 题目分类

我将题目按照一些常见的算法类别进行了分类，方便查找和学习不同类型的题目。

- 数组与字符串
- 链表
- 栈与队列
- 哈希表
- 排序与搜索
- 贪心算法
- 动态规划
- 图与树
- 位运算
- 设计问题
- 数学问题
- ...
